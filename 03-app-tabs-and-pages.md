# App Tabs And Pages

## Bottom Tabs

### Home

The main shopping entry point. It shows stories, live sessions, trending products, personalized products, and search. Buyers can open products and livestream sessions directly from this tab.

Key features:

- Story ring
- Live Now section
- Trending products
- For You product recommendations
- Location-aware discovery
- Product search entry

### Feed

Social commerce feed for browsing seller posts, reels, local content, and followed seller content.

Key features:

- For You feed
- Following feed
- Reels feed
- Local feed
- Like and unlike posts
- Save and unsave posts
- Follow and unfollow sellers
- Product-tagged content

### Inbox

Realtime buyer-seller communication and alerts.

Key features:

- Conversation list
- Direct seller conversations
- Realtime messages
- Typing indicators
- Read receipts
- Message reactions
- Order updates
- Alerts and notifications

### Orders

Order history and order management. This tab is configured as an internal tab screen and can be opened from account or related flows.

Key features:

- Active, delivered, and cancelled order views
- Detailed order data
- Cancel order
- Reorder
- Submit rating or review

### Wishlist

Saved product management. This tab is configured as an internal tab screen and can be opened from account, product, or saved item flows.

Key features:

- Wishlist item list
- Wishlist collections
- Create collection
- Save product into a collection
- Move from wishlist to cart
- Remove saved product

### Settings

Account, preference, seller, privacy, and support area.

Key features:

- Profile and account dashboard
- Theme preference
- Address management
- Seller onboarding
- Security settings
- Push notification registration
- Data export request
- Legal and app information

## Authentication Pages

### Login

Supports email/password login and Google sign-in. Stores access and refresh tokens securely.

### Register

Creates a new account and starts the email verification flow.

### Email Verification

Accepts verification code and activates the user's email status.

### Forgot Password

Requests password reset email.

### OAuth Redirect

Handles OAuth deep-link return behavior for sign-in flows.

## Shopping Pages

### Product Detail

Detailed product page with images, seller info, price, stock, delivery information, reviews, wishlist action, cart action, and similar products.

### Cart

Shows selected items, quantity controls, item removal, subtotal, and checkout entry.

### Checkout

Collects delivery address, payment method, coupon, order summary, and starts Razorpay or cash-on-delivery flow.

### Order Confirmed

Post-checkout confirmation screen with order success state and next-step navigation.

### Order Detail

Detailed order status, payment state, item summary, cancellation, and tracking-related information.

## Seller Pages

### Seller Profile

Public seller storefront with seller stats, products, posts, stories, live sessions, reviews, and follow action.

### Seller Dashboard

Seller workspace for managing business operations.

Key features:

- Revenue and sales overview
- Product performance
- Order management
- Product create, update, and delete
- Wallet withdrawal request
- Review replies and report flow
- Push notification registration
- Recent activity and alerts

## Content Creation Pages

### Create Post

Seller or creator flow for creating product-linked posts and media content.

### Create Story

Story creation flow with media selection and story metadata.

### Story Viewer

Full-screen story consumption with views, reactions, and viewer data.

### Saved Posts

Saved social content area for posts saved from the feed.

## Livestream Pages

### Livestream Viewer

Buyer livestream experience.

Key features:

- Agora video viewer
- Realtime chat
- Reactions
- Pinned product strip
- Buy Now modal
- Q&A updates
- Session end handling

### Livestream Host

Seller livestream control view.

Key features:

- Host camera stream
- Chat moderation tools
- Pin product
- Pin message
- Toggle Q&A
- Answer questions
- End session

## Inbox Detail Pages

### Conversation

Single chat thread between buyer and seller.

Key features:

- Message history
- Realtime send and receive
- Typing status
- Read state
- Reactions
